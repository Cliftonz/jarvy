Devfile = YAML text file recording instructions to configure and run a build environment. Adopted from public community registry or authored locally. Supported by build tools and IDEs that auto-process the instructions.

Capabilities given to tools/IDEs:
- Pull in repo hosting application source
- Build the code
- Run the application in a local container
- Deploy to cloud-native containers

Community: GitHub repo devfile/api for issues/questions.
