# Benefits

- Workspaces composed of multiple containers
- Create N identical workspaces from one devfile
- Share a devfile across teams for same UX + build/run/deploy
- Features: runtime image guidance, example code, build/CI commands, deployment options
- Benefits: reduce dev-to-deploy gap, find stacks/samples in registry, consistent build/run

# Audiences
- Application developers
- Enterprise architects / runtime providers
- Registry administrators
- Technology / tool builders
