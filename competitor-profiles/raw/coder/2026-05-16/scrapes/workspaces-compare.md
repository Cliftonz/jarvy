# Coder vs. competitors (their framing)

Positioning: "self-hosted, open-source AI development infrastructure platform with built-in governance for enterprise teams"

## vs GitHub Codespaces
Open source, self-hosted, infrastructure-agnostic. Avoid vendor lock-in. Supports all IDEs not just VS Code Web. Terraform RBAC.

## vs Gitpod
Fully self-managed vs Gitpod managed-only and AWS-locked. Better for air-gapped, deep internal integration.

## vs AWS CodeCatalyst
Multi-cloud / hybrid / on-prem vs CodeCatalyst's AWS lock-in.

## vs Microsoft Dev Box
Container/VM lightweight envs spin up in seconds vs VDI Windows desktop. Lower licensing/infra cost. Cross-OS support.
