# Pricing

## Community — Free
For hobbyists and small teams. Open source.
Install via `coder.com/docs/install`.

Includes:
- Unlimited workspaces and templates
- Unlimited members within a single organization
- Desktop and web IDEs (Cursor, VS Code, JetBrains, IntelliJ, Jupyter, more)
- Web UI, CLI, REST API
- Long-running tasks for AI coding agents
- SSO (OIDC)
- Community support (Discord, GitHub)
- Limited to 1 platform integration
- Limited to 1 external auth integration

## Premium — "Annually per user" (exact price not published, contact sales)
For enterprises.
Trial via `coder.com/trial`.

Adds:
- Ticket-based global support with SLA
- Multi-organization access controls
- Resource quotas per organization and user
- Audit logging
- Unlimited git + external auth integrations
- High availability (multiple Coder server replicas)
- Workspace proxies (low-latency global relays)
- Auto-stop idle workspaces
- OIDC group/role sync
- Group + user RBAC
- Custom branding
- AI Governance add-on available (Agent Firewall, AI Gateway as add-ons)

Compare matrix categories: Agent Infrastructure (MCP Server, Agent Firewall add-on, AI Gateway add-on), Developer Experience, Platform Experience, User Management, Cloud Cost Control, Governance, Support.

Logos on pricing page: Square, Dropbox, KKR, Palantir, Mercedes
