# Pricing (Ona)

Unit: Ona Compute Units (OCUs) — covers both agent token use + environment runtime.

## Free
- $10 of free usage, 40 OCUs one-time
- 4 cores / 16GB RAM / 80GB disk
- VS Code Web + one-click desktop IDE
- Up to 3 parallel environments
- User + project secrets, MCP server support
- Environments auto-delete after 3 days inactivity
- Background Agents access
- 1 seat

## Core — from $20/month
- 80-2,200 OCUs/month
- Add-on OCUs from $10 / 40 OCUs
- Up to 100 team members (pooled OCUs, not per-seat)
- Unlimited parallel environments
- Up to 32 vCPUs / 128GB RAM / 200GB disk
- GPU support (up to 16 vCPUs / 64GB / 300GB)
- Prebuilds, environment lifecycle mgmt
- Project sharing, RBAC
- Org-wide commands + deny lists
- 7-day inactivity auto-delete
- 50 automations / 10 parallel actions / 50 total actions

## Enterprise — custom pricing
- Custom OCU credits
- Self-hosted in customer VPC, Ona-managed (AWS or GCP)
- Bank-grade VPC deployment
- Full network control (load balancers, domain, certificates), HTTP proxy support
- Warm pools (instant starts)
- Fine-grained roles, policies, SSO + OIDC
- Org-wide secrets
- Audit trails (every human + AI action)
- Org-wide engineering productivity insights
- Control over MCP usage
- Programmatic SDK + API access
- SLA, dedicated account manager + forward-deployed engineer

OCU usage examples:
- Run 4vCPU/16GB VM 1hr = 1 OCU
- Run GPU VM (16vCPU/64GB) 1hr = 7 OCUs
- Explain small codebase = 1 OCU
- Fix bug in large codebase = 5 OCUs
- Add feature to medium codebase = 8 OCUs

Compliance: GDPR, SOC 2, Fortune 500 references.

OCU expiry: free credits 1 yr, monthly sub credits end of billing month, add-ons 1 yr.
No training on customer code.
