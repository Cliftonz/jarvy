## PRD-004: Semantic Version Checking

**Files**: `src/tools/version.rs` (new), `src/tools/common.rs`
**Dependencies**: semver, regex, once_cell
**Tests**: 24 unit tests

**Key Features**:
- Semver operators: `>=`, `<`, `^`, `~`, `=` and ranges
- Smart version extraction from tool output (git, node, python, docker, rustc, go)
- Component-boundary matching prevents false positives ("3.1" won't match "3.10.0")

**Key Learnings**:
- `VersionReq::parse("3.10.5")` returns `^3.10.5`, not exact match - must use `=3.10.5`
- Regex pattern: `v?(\d+)\.(\d+)(?:\.(\d+))?(?:-([a-zA-Z0-9.-]+))?`

## PRD-009: Service Management

**Files**: `src/services/` (mod.rs, docker_compose.rs, tilt.rs)
**Tests**: 25 unit tests + 8 integration tests

**Key Features**:
- `[services]` config section with `enabled`, `auto_start`, `compose_file`, `tilt_file`, `start_in_ci` options
- Two backends: Docker Compose and Tilt (via `ServiceBackend` enum)
- `ServiceBackendOps` trait for polymorphic backend operations (start, stop, status)
- `jarvy services start/stop/status/restart` CLI commands
- Auto-start integration in setup flow (respects CI mode)
- Detection priority: Docker Compose over Tilt when both present
- Graceful handling when docker/tilt not installed

**CLI Structure**:
```
jarvy services -f <config> start [--foreground]
jarvy services -f <config> stop
jarvy services -f <config> status
jarvy services -f <config> restart [--foreground]
```

**Key Learnings**:
- Clap CLI argument ordering: parent command flags (`-f`) must come before subcommands
- Test assertions should check both stdout and stderr when backend may not be installed
- Advisory error handling: service failures warn but don't fail setup
- `should_auto_start(is_ci)` pattern cleanly handles CI mode logic

## Key Technical Learnings

1. **Rust Macros**: Helper patterns (`@macos`, `@linux`) make complex `macro_rules!` manageable
2. **Inventory Crate**: Compile-time static collection via linker - zero runtime cost
3. **Serde Flatten**: `#[serde(flatten)]` for capturing dynamic TOML keys
4. **Idempotent Scripts**: `grep -q` pattern before shell rc modification
5. **Semver Gotcha**: Bare versions default to caret (^), not exact match
6. **Rust 2024**: `set_var`/`remove_var` now unsafe due to race condition potential
7. **Path Quirks**: `with_extension()` behaves unexpectedly on dotfiles
